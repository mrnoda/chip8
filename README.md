# chip8
A Chip-8 simulator, written in C, using SDL2

Current state:
  * Full CPU emulation
  * Audio beep (though this currently delays execution by approx 500ms :F)
  * SDL graphics (scaled up to 640x480 resolution)
  * Keyboard input
